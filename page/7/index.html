Redis（十）哨兵模式

发表于 2020-10-31 更新于 2022-09-22 分类于 Redis

为什么需要哨兵？

当主服务宕机后，需要手动把一台从服务器切换为主服务器，这就需要人工干预，费时费力，还会造成一段时间内服务不可用，这不是一种推荐的方式

Redis从2.8开始正式提供了Sentinel（哨兵）架构来解决这个问题。它能够后台监控主机是否故障，如果故障了根据投票数自动将从库转换为主库。
